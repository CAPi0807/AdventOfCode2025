# Advent of Code 2025

## Objetivo del Proyecto
Este proyecto alberga las soluciones para el [Advent of Code 2025](https://adventofcode.com/2025). El propósito principal es resolver los retos de programación diarios, poniendo a prueba y mejorando habilidades en algoritmos, estructuras de datos y lógica de programación utilizando Java.

## Enfoque Técnico
El proyecto está estructurado para ser modular y fácil de navegar:

*   **Lenguaje**: Java.
*   **Organización**: Cada día del calendario tiene su propio paquete (`software.aoc.dayXX`).
*   **Partes**: Dentro de cada día, las soluciones se dividen típicamente en parte `a` y parte `b` para abordar las dos mitades del problema diario.
*   **Documentación**: Cada día cuenta con su propio `ReadMe.md` que detalla el problema específico y la solución implementada.
